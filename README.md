# intern-ruby-test
Ruby app with React UI

# Requierments

- ruby 2.6.3p62
- node v10.16.0
- npm 6.9.0 | yarn 
- sqlite3

Easy env setup
- rvm
- nvm

# How To Run

Server

`git clone https://github.com/adevofficial/intern-ruby-test`\
`cd intern-ruby-test/api`\
`bundle install`\
`rails db:migrate`\
`rails s`

Client

`cd client`\
`yarn` or `npm install`\
`yarn start` or `npm start`
